internet-date.com
A dating app with user profiles, allows users to upload content to a public and private feed. 

Site home page is the public feed. Public feed is mix of user submitted content and dating matches. 

Site consists of: site feed (homepage), user profiles, quiz taking page, message inbox.

Functionality:

Site Feed (Homepage)
  Infinite scrolling feed
  Public user posts (text, image, video)
  Match suggestion cards
  Like / pass actions
  Save post action
  Feed filter toggle (posts / matches)
  Logged-out read-only view
  Signup / login CTA

Quiz Taking Page
	Single active quiz
	Multiple-choice questions
	Scale-based questions
	One-question-per-screen flow
	Progress indicator
	Quiz completion submission
	Quiz results stored to profile
	Quiz results used for match scoring

Message Inbox
	Match-only conversations
	Inbox list with latest message preview
	Unread message count
	One-to-one chat threads
	Text message sending
	Message read status
	User block action
	User report action

User Profiles
	Photo and video internal feed
	Public and private post visibility per item
	Short bio
	Age display
	Location display
	Dating intent display
	Profile rating widget (attractive, smart, funny, interesting)
	User Top 5 lists
	Owner-only profile edit controls
